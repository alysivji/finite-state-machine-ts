# Examples

These examples mirror the documented state machines covered by the example test suite. Each page includes:

- The Mermaid state diagram for the machine.
- A complete TypeScript example.
- A short explanation of how the transition methods map to the diagram.

Core sync/async transition semantics are covered separately by focused unit tests, so these examples stay small and documentation-oriented.
